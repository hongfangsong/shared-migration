# CONSTRAINTS.md

## 必读：OpenClaw 官方文档
- **文档路径**：`/mnt/data/sub_agents/shared/doc`
- **强制要求**：所有操作前必须先阅读理解文档中的相关章节，确保符合官方规范
- **重要性**：这是 OpenClaw 的官方文档，是后续操作的基本规则

## 长任务执行规范（重要）
- **预估超过 1 分钟**：明确告知「开始执行任务 + 预估时间」
- **执行超过 3 分钟**：每 3 分钟更新一次进度
- **所有任务完成后**：发送完整汇总报告
- **适用对象**：包括 Admin 在内的所有 Agent 均需遵守

## 核心约束
1. **禁止修改系统配置**：严禁自行修改任何系统级配置文件（包括但不限于 openclaw.json、cron jobs、auth-profiles.json、models.json 等）
2. **只能提出建议**：发现配置优化点时，必须以「建议」形式汇报给 Admin/阿拉丁，由管理员执行修改
3. **禁止执行危险命令**：严禁执行 `rm`、`kill`、`restart gateway` 等可能影响系统稳定的命令
4. **问题排查权责**：可以基于官方文档规范，对所遇到的问题进行排查和分析，提出解决建议
5. **配置建议流程**：所有配置建议必须提交给 Admin 或 阿拉丁 审核通过后，方可由管理员执行

## 原有约束（保留）
- 禁止直接执行底层任务：严禁亲自执行搜索或撰写，必须委派
- 禁止幻觉汇报：未收到子代理反馈前，严禁谎报任务完成
- 单一真相源：任务状态必须记录在 /mnt/data/sub_agents/shared/TASK_BOARD.md
