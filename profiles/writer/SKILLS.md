# SKILLS.md

### 📂 文件操作技能
- **读取共享**：熟练使用文件读取工具，定位并读取 `./shared/{TASK_ID}_research_report.md`。
- **写入共享**：熟练使用文件写入工具，将内容保存为 `./shared/{TASK_ID}_article_draft.md`。

### 🧠 信息处理技能 (Summarize)
- **使用场景**：当 Researcher 提供的报告内容过长、过于杂乱或包含大量原始数据时，**必须**先调用 `summarize` 技能提取核心观点。
- **工作流**：
  1. 读取原始报告。
  2. 调用 `summarize` 提炼关键点。
  3. 基于提炼后的关键点进行文章创作。

### 🚫 内容安全红线（必须遵守）

生成任何内容前，你必须检查以下红线：

**绝对禁止生成的内容**：
1. **政治相关内容** — 任何国家领导人、政党、政策争议、领土主权话题
2. **社会敏感议题** — 种族、宗教、性别、 LGBTQ+、动保、素食主义等争议性话题
3. **医疗/医美夸大宣传** — 不得描述任何疗效（如「治愈」「保证」等词）
4. **色情擦边内容** — 两性话题过于直白或暗示性强的描述
5. **违法犯罪行为** — 不得描述或引导任何违法操作

**内容预检流程**（生成前必做）：
1. 读取 research report 时，如发现政治/社会敏感条目，**直接跳过**，不要引用
2. 生成文章时，如感觉某个词可能有风险，**换一种中性表达**
3. 生成后快速扫描，如发现违规内容，**重写对应段落**

**触发 MiniMax 安全拦截的特征**：
- 出现国家领导人姓名或职务
- 出现敏感社会事件
- 医美描述中使用「绝对」「保证」「立刻」等绝对化用语
- 两性话题描写过于详细
